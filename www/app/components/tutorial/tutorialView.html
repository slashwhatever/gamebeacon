Thank you for helping to test gamebeacon for Destiny!

The app is designed to help you find a group of like-minded guardians for raids, crucible action or general missions, all from your mobile device.

The app is still in it's Alpha phase.

This means you will find bugs.

Please tell us about them by emailing us.

Getting Started

When you login you should see a list of active Beacons

Each Beacon Card in the list has a set of icons to give you a quick heads up about the type of game people want to play.

In the Beacon Card above you can see the region, platform, level, number of Guardians still needed and whether a mic is required.

The little yellow flash in the top right corner signifies this is your Beacon - one you created.

Beacon Detail

When you tap through to view a Beacon, you'll be able to join, chat with other users and see the users who've already joined.

If you created the Beacon, you have the power to delete it or to kick Guardians from the Beacon (click their user name to see the action menu).

Other Stuff

If there are no Beacons you like the look of, create your own.

The action button will help you do this.

You can only join one Beacon or create one Beacon of your own at any given time.

Beacons auto-expire after 30 minutes.

If you created the Beacon, you have the power to delete it or to kick Guardians from the Beacon (click their user name to see the action menu).

That's It!

That's really all there is to it. Go create a Beacon!
