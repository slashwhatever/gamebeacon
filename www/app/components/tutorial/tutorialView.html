Getting Started

Each gamebeacon in the main view has a set of icons to give you a quick heads up about the type of game people want to play.

Above you can see a couple of gamebeacons that show the region, platform, level, number of Guardians still needed and whether a mic is required for each session.

The little yellow flash in the top right corner signifies this is your gamebeacon - one you created.

Swipe the calendar strip above the gamebeacon list to see gamebeacons starting on different days.

Next Steps

When you tap through to view a gamebeacon, you'll be able to join, chat with other users and see the users who've already joined.

If you created the gamebeacon, you have the power to delete it or to kick Guardians (click their name to see the action menu).

Other Stuff

If there are no gamebeacon you like the look of, create your own.

The action button will help you do this.

That's It!

That's really all there is to it. Go create a gamebeacon!
