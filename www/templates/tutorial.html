Thank you for helping to test the game beacon app for Destiny!

The app is designed to help you find a group of like-minded guardians for raids, crucible action or general missions, all from your mobile device.

The app is still in it's Alpha phase.

This means you will find bugs. Please tell us about them using the feedback form!

Getting Started

To get started, you'll need to ether create a Beacon or join someone elses.

The action button in the bottom right of the screen will help you do this.

Beacon List

Each Beacon in the list features a set of icons to give you a quick heads up about the type of game people want to play.

In the beacon card above you'll see the region, platform, level, number of Guardians still needed and whether a mic is required.

As more Guardians join a beacon, the number of Guardians will reduce until no-one else can join.

Beacons auto-expire after 30 minutes.

Beacon Detail

When you click through to view a Beacon, you'll be able to join the Beacon and chat about the game you're about to play.

You can only join one Beacon or create one Beacon of your own at any given time.

If you created the Beacon, you have the power to delete it or to kick Guardians from the Beacon (click their user name to see the action menu).

That's It!

That's really all there is to it. Go create a Beacon!
